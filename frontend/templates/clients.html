Como Usar a API
Buscar lista de domínios (TXT)
curl -H "X-API-Key: SUA_API_KEY" \
  {{ request.host_url }}api/v1/client/domains?format=txt
Buscar lista de domínios (RPZ/Unbound)
curl -H "X-API-Key: SUA_API_KEY" \
  {{ request.host_url }}api/v1/client/domains?format=rpz
Verificar status
curl -H "X-API-Key: SUA_API_KEY" \
  {{ request.host_url }}api/v1/client/status
Confirmar sincronização
curl -X POST \
  -H "X-API-Key: SUA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"domains_applied": 1234, "status": "success"}' \
  {{ request.host_url }}api/v1/client/sync/complete
